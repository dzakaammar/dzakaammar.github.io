Creational Pattern: Singleton

Singleton menyediakan cara untuk membuat single instance dan menyediakan cara untuk mengakses/menggunakan instance tersebut sebagai global variable. Memastikan hanya terdapat satu instance dari produk tertentu selama aplikasi berjalan. Karakterstik Terdapat global variable untuk mendefine single instance. Contoh tipe produk: meja, kursi, sofa.
